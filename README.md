# Graham-Buffett Investment Agent — Desktop GUI

A point-and-click investment research tool. No command line needed.

## How to start

**Windows:** Double-click `START.bat`

That's it. Your browser opens automatically at http://localhost:3000

## First-time setup (inside the app)

1. Choose your **AI Provider** (Gemini, Claude, or OpenAI)
2. Choose a **Model** from the dropdown
3. Paste your **API key**
4. Click **Save Settings** — your key is saved locally for next time
5. Type a ticker and press Enter (or click ▶)

## Where to get API keys

| Provider | Free tier | Link |

**Gemini is recommended** if you want to start for free.

## Stopping the app

Close the black Command Prompt window that opened with START.bat.
